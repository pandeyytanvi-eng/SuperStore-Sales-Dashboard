# 📊 SuperStore Sales Dashboard

## Overview

This project is an interactive Power BI dashboard built using the SuperStore Sales dataset. The dashboard provides meaningful insights into sales performance, profitability, customer behavior, and regional trends. It helps users analyze business performance through interactive visualizations and filters, making data easier to understand and explore.

---

## Features

- Sales and Profit Analysis
- Region-wise Performance
- Category and Sub-Category Analysis
- Customer Segment Insights
- Monthly Sales Trends
- Interactive Filters and Slicers
- KPI Cards for quick business insights

---

## Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

## Dataset

The dashboard is created using the **SuperStore Sales Dataset**, which contains information related to orders, customers, products, sales, profits, shipping, and regions.

---

## Dashboard Insights

The dashboard helps answer questions like:

- Which region generates the highest sales?
- Which product categories are the most profitable?
- How do monthly sales change over time?
- Which customer segment contributes the most revenue?
- Which products perform well and which need attention?

## How to Use

1. Download the repository.
2. Open `superstore sales.pbix` in Microsoft Power BI Desktop.
3. If prompted, reconnect the Excel dataset.
4. Refresh the data to view the latest dashboard.
